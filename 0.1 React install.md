## Vite vs Create React App

| Feature | Vite | CRA |
|---------|------|-----|
| **Startup speed** | ⚡ Very fast | Slow |
| **Bundler** | ESBuild | Webpack |
| **Hot Reload** | Instant | Slower |
| **Modern standard** | ✅ Yes | ❌ Deprecated |
